# TODOタスク参照API

## API IF

形式：json

- リクエスト
  - id: タスクのID
- レスポンス（成功時）
  - id
  - title
- レスポンス（エラー時）
  - errorMessage 

## DB

TODO_TASK

| カラム | 説明 |
| --- | --- | 
| id | PK。タスクを一意に特定するもの |
| title | タスクのタイトル |

## 処理内容

1. リクエストのIDでTODO_TASKからデータを取得する
2. 取得したタイトルに「削除」が含まれる場合は、そのタスクを除外し、データなしエラーを返却する
3. 取得したタスクをレスポンスとして返却する
